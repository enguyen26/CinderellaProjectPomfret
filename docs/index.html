The Cinderella Project

A mission-driven web app that helps students browse donated formal dresses, view impact stories, and use AI styling support to feel confident for their big event.

Quick Pitch

  • Problem: many students cannot afford formalwear.
  • Solution: accessible catalog + filtering + impact storytelling.
  • Innovation: AI assistant for personalized outfit guidance.
  • Design: warm editorial style to match the mission.

Demo Flow (What I Will Show)

  1. Landing page mission, tone, and user trust design choices.
  2. Gallery section showing community impact and social proof.
  3. Catalog filters by size, style, and availability for accessibility.
  4. Virtual try-on upload flow and AI-generated styling response.
  5. Modal details and reusable UI components.

Technical Highlights

  • State-driven tabbed navigation in React.
  • Data filtering logic for real-time catalog updates.
  • File upload + image preview using FileReader.
  • External API request pipeline for AI stylist output.
  • Component-level styling tuned to brand identity.
App file: cinderella-project/cinderella-project.jsx

What Makes It Feel Human-Made

  • I chose a consistent charity-focused voice, not generic app text.
  • The UI balances polish and warmth with intentional color palette choices.
  • Feature decisions map directly to real user needs and constraints.
  • The pitch and flow are story-first, not just listing technical buzzwords.

Future Improvements

  • Authentication for student/admin dashboard roles.
  • Real backend inventory system with reservations and pickup scheduling.
  • Accessibility audit (keyboard navigation and ARIA labels).
  • Usage analytics to improve donation and request matching.
